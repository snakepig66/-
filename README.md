随着物联网技术向“智联”方向演进，传统零售业的数字化转型面临算力成本与边缘设备性能之间的矛盾。针对传统超市生鲜称重环节存在的效率低下、培训成本高及易作弊等痛点，本文设计并实现了一套基于RTOS（实时操作系统）与“云-边-端”协同架构的智能视觉称重系统。本系统创新性地采用了双ESP32-S3微控制器的分布式边缘视觉架构，有效解决了图像采集与网络传输争抢CPU资源的问题。在软件层面，系统基于深度定制的FreeRTOS，实现了多任务并发环境下的毫秒级响应；在算法层面，系统通过调用云端深度学习API实现高精度果蔬识别，并融合了QMI8658姿态传感器与卡尔曼滤波算法，实现了秤体倾斜防作弊监测。测试结果表明，该系统突破了单一触控模式，成功融合了机器视觉、智能语音与MQTT移动端遥测等多模态交互，在低成本嵌入式硬件上实现了极高的识别准确率与稳定的云边协同计算，为智慧零售终端的普及提供了新范式。
关键词：智能称重；机器视觉；FreeRTOS；云边协同；多模态交互
